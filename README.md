## How to setup and run 
1. Create .env file 
```
PORT = [PORT]
DB = postgres://postgres:[PASSWORD]@[HOST]/[DATABASE_NAME]?sslmode=disable
```  
2. Import database from folder database  
3. Run by type  
`go  run ./cmd/api/. ` or `go run cmd/api/*`
